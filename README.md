Compile with:
```sh
$ make
```

Run client with:
```sh
$ ./SimpClient http://www.google.ca/index.html
```
OR
```sh
$./SimpClient
```
to get a prompt to enter URI

Run server with:
```sh
$ ./SimpServer portNumber nameOfdirectory
```
